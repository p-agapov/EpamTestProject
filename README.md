# Project description
Current system is meant to provide to users an opportunity for ordering vacation tours.

# Acting roles
+ User
+ Tour Manager (admin)

# Entity

# Tech stack
+ Programming language - Java 8
+ DB - PostgreSQL
+ Test framework - JUnit 5
+ Build tool - Maven
